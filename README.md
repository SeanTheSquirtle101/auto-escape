<h1 align="center">Auto Escape</h1>

### A simple mod to automatically quit to the main menu when a Bloon would otherwise leak and kill you.

You won't be Black Bordering any maps with this, but you should at least save the Monkey Money on Continues.

## NOTE: This alpha version of the mod requires [BTD6 Mod Helper >v3.0](https://github.com/gurrenm3/BTD-Mod-Helper/wiki/Mod-Helper-3.0-Alpha), see [here](https://github.com/doombubbles/BTD6-Mods#readme) for the stable mods
